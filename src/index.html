There are two example spreadsheets here. Handsontable and AG Grid
Handsontable seems to be the most well used and fully featured but it is expensive for commercial use
AG grid seems to be the most accessible of the libraries. It actually has a page on accessibility and the accessibility tools show it is mostly accessible.
The spreadsheet for both uses the same data. There are buttons to display the spreadsheet data and buttons to load data from an external source which will be useful for things like infinite scroll.
There are other features

  • You can change the price value below 10000 and the cell will go red
  • The make column is a drop down.
  • All columns can be filtered and sorted
